## Phương pháp học: "Tay cầm tay có kiểm soát"

### Quy trình mỗi bài:

1. AI đưa code mẫu (5-15 dòng) + giải thích ngắn
2. Người học **tự gõ lại** (không copy paste)
3. Người học **sửa đổi** (đổi tên biến, thay đổi giá trị, thêm chức năng)
4. Người học **tự viết lại từ đầu** (đóng code mẫu)
5. Người học làm bài tập tương tự

### Nguyên tắc sử dụng AI:

- ✅ Hỏi khái niệm, lý thuyết
- ✅ Hỏi hướng giải quyết
- ✅ Nhờ review code
- ❌ Không copy code từ AI
- ❌ Không để AI viết code hộ
- ❌ Không để AI sửa lỗi hộ

### Khi gặp khó:

1. Google 5 phút
2. Tự thử 10 phút
3. Hỏi AI hướng (không code mẫu)

## Lộ trình 4 tháng

### Giai đoạn 1: JavaScript nền tảng (Tháng 1)

- Biến, kiểu dữ liệu, if/else, vòng lặp
- Array, object, function
- Array methods: forEach, map, filter, find, reduce
- Callback, Promise, async/await
- try/catch, error handling

### Giai đoạn 2: Node.js + Express + Database (Tháng 2)

- Node.js: require, module, fs, npm
- Express: route, middleware, error handling
- MongoDB: Schema, Model, CRUD
- Project 1: Todo API
- Project 2: Blog API

### Giai đoạn 3: Authentication + Nâng cao (Tháng 3)

- Bcrypt, JWT
- Register + Login API
- Project 3: Auth API
- Project 4: E-commerce API
- Deploy lên Render

### Giai đoạn 4: Luyện phỏng vấn (Tháng 4)

- Ôn lý thuyết
- Câu hỏi phỏng vấn
- Mock interview
